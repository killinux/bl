基础操作

4 个 API · 最常用
核心提示：创建物体、修改材质、打灯光、设置相机、渲染输出、设置动画关键帧等操作都通过 execute_blender_code() 执行 bpy 代码实现，没有单独的高层 API。
get_scene_info() 获取场景全部信息 ▶

参数

无参数

返回

场景名称、所有对象列表（名称、类型、位置、旋转、缩放）、活动相机、渲染设置等

用法示例

列出当前 Blender 场景中的所有对象

参数

名称类型说明

返回

位置、旋转、缩放、材质列表、网格数据（顶点数/面数）、包围盒、修改器等

用法示例

获取 Cube 的详细信息，包括它的材质和顶点数
execute_blender_code(code) 执行任意 Python 代码 ▶

参数

名称类型说明
codestring要执行的 Python/bpy 代码

返回

代码执行的标准输出、错误信息

用法示例

# 创建一个红色球体 import bpy bpy.ops.mesh.primitive_uv_sphere_add(radius=1, location=(0, 0, 0)) obj = bpy.context.active_object mat = bpy.data.materials.new("Red") mat.use_nodes = True mat.node_tree.nodes["Principled BSDF"].inputs["Base Color"].default_value = (1, 0, 0, 1) obj.data.materials.append(mat)
万能 API：这是最强大的工具。创建物体、修改材质、灯光、相机、渲染、动画、导入导出等一切操作都可以通过它完成。

blender-mcp vs Bridge 方案对比

blender-mcpBridge 方案
网络要求同一台机器 (localhost)可跨网络，内网穿透
集成方式MCP 原生工具CLI 命令行
核心能力执行代码 + 截图 + 场景查询执行代码 + 截图 + 日志 + 文件推送
资产库PolyHaven + Sketchfab无（可通过代码执行间接使用）
AI 生成Hyper3D + Hunyuan3D无
适用场景本地开发远程开发（AWS + 内网 Mac）
